# Mood Analyzer

Mood Analyzer is a lightweight Python-based system that analyzes daily mood logs, detects emotional patterns, identifies mood trends, clusters similar notes using embeddings, and generates safe micro-actions.

## Features
- Daily mood parsing and mapping to numerical scores
- Temporal pattern detection (weekday mood dips/peaks)
- Clustering of notes using SentenceTransformer embeddings
- Confidence scoring for detected patterns
- Microâ€“actions generation (empathy-safe)
- Matplotlib mood trend visualization
- Jupyter Notebook interactive demo
- Unit tests for pattern detection
- Fully modular Python project structure
